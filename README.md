# Arduino Distance Parking System 🚗

## 📌 Overview
This project is an Arduino-based system that helps drivers park safely using a distance sensor.

## ⚙️ Components
- Arduino board
- Ultrasonic distance sensor (HC-SR04)
- LEDs
- Buzzer
- Jumper wires

## 💡 How it works
The system measures the distance between the car and an object:
- Green LED → Safe distance
- Yellow LED → Getting close
- Red LED + Buzzer → Too close

## 🔌 Circuit
Connect the ultrasonic sensor to the Arduino and attach LEDs and buzzer to output pins.

## ▶️ Usage
1. Upload the code to Arduino
2. Power the system
3. Move object closer/farther to test
